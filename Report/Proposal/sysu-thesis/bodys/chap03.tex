\section{预期成果}

本项目的预期交付物包括：

\begin{itemize}
    \item 一个集文档解析、视觉定位与自然语言理解于一体的“文档智能助手”系统原型，支持检索问答与引用溯源。
    \item 相关的代码实现与技术文档，包含模型训练、系统集成、部署与测试说明。
    \item 一份完整的项目报告，包括实现的具体细节以及定量的实验结果与可视化，以及反思与展望。
    \item 演示材料与最终报告，包括 PPT、Demo 展示用于期末验收
\end{itemize}
